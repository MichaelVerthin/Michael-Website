Project "Mobile"

פרוייקט טלפון

The third and final project that we had to make under the Quality Assurance course that I took, This time around we had to make an STR about issues that we find in any application that we choose! so the bar is set even tighter then last time around since mobile apps are usually more robust than some websites

הפרויקט השלישי והאחרון שהיינו צריכים לעשות במסגרת קורס הבטחת איכות שלקחתי, הפעם היינו צריכים לעשות STR על נושאים שאנו מוצאים בכל אפליקציה שנבחר! כך שהרף מוגדר הדוק עוד יותר מהפעם הקודמת מכיוון שאפליקציות לנייד בדרך כלל חזקות יותר מאתרים מסוימים
